# Miniviewer

A simple image visualizer written in [C](https://en.wikipedia.org/wiki/The_C_Programming_Language) and [SDL2](https://www.libsdl.org/).

# Formats

It can open only:
JPG, PNG, TGA, BMP, PSD, GIF, HDR & PIC.

# Quick Start
```console
$ make
$ ./miniviewer ./test.png
```
